Mac使用ngrok内网穿透

# 注册ngrox

建议使用github账号注册。注册地址 (opens new window)

# 下载客户端

进入下载页面 (opens new window) 选择自己需要的版本

# 解压ngrok

可以使用电脑上的解压软件或者命令行进行解压

unzip /path/to/ngrok.zip
1

# 添加账号认证

认证信息在下载页面可以查看

ngrok config add-authtoken XXXXXXX
1

# 开启ngrok

ngrok http 8080
1

注

如果出现-bash: ngrok: command not found,将ngrok放置到/usr/local/bin/可解决，操作如下

sudo cp -r xx/xx/ngrok /usr/local/bin/
1
